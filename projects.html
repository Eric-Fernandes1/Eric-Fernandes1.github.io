6D Pose Estimation System

Challenge

Enable real-time 6D object pose estimation for variable orientations and integrate it with an ABB manipulator for adaptive pick-and-place without fixed waypoint programming.

Technologies

  • Intel RealSense RGB-D sensing
  • YOLO-based detection + custom PnP solver
  • Camera-to-robot calibration (hand–eye transformation)
  • Python trajectory generation with ABB RAPID socket communication

Impact

  • Sub-centimeter pose accuracy
  • Eliminated manual re-teaching between cycles

ABB Robot Control via OPC UA

Challenge

Develop secure and reliable communication between ROS2-based perception system and ABB industrial robot without using EGM.

Technologies

  • ROS2 communication framework
  • OPC UA Gateway integration
  • TCP socket-based data exchange
  • Industrial PLC interfacing

Impact

  • Secure industrial communication architecture
  • Improved modularity and scalability
