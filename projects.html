6D Pose Estimation System

Developed a real-time 6D pose estimation pipeline using Intel RealSense camera, YOLO-based detection, and a custom PnP solver. The system was integrated with an ABB industrial robot for autonomous pick-and-place operations.

ABB Robot Control via OPC UA

Designed and implemented industrial robot control architecture using ROS2 and OPC UA gateway without using EGM. Ensured reliable and secure communication between robot, PLC, and vision system.

YOLO + ArUco Pose Pipeline

Combined deep learning object detection with ArUco-based pose refinement to improve accuracy in robotic manipulation tasks. Implemented coordinate transformation pipeline from camera frame to robot base frame.

Bin Picking Simulation (Simulink)

Developed a perception-control-planning pipeline in MATLAB/Simulink including deep learning-based object pose estimation integrated with motion planning algorithms.
